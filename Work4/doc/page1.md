1.2常见的视觉传感器
通常可以分为两种：
○普通 RGB 相机 是机器人的“眼睛”，让机器人“看见”；
○RGB-D 深度相机 是机器人的 “带测距功能的眼睛”，让机器人不仅 “看见”，还能 “感知” 世界的三维结构。
在实际应用中，我们会根据具体任务需求选择合适的传感器，或结合使用以获得更全面的环境感知。

1.3普通 RGB 相机（Camera）
●功能：获取场景的二维彩色图像。
●原理：基于针孔成像模型，捕捉物体反射的可见光，形成与人眼视觉相似的彩色画面。
●数据输出：sensor_msgs/Image 消息，包含高度、宽度、像素数据（如 BGR8 格式）。
●优点：结构简单、成本低、分辨率高、色彩信息丰富。
●缺点：无法直接获取物体到相机的距离。
●用途：
○目标识别与分类（如人脸识别、二维码识别）；
○视觉伺服（如引导机械臂抓取已知位置的物体）；
○颜色追踪、视觉里程计（VO）的基础输入。

1.4RGB-D 深度相机
●功能：同时获取场景的二维彩色图像和三维深度图像。
●原理：通过结构光、飞行时间法（ToF）或双目视觉等技术，主动或被动地测量每个像素点到相机的物理距离。
●数据输出：
○彩色图像：sensor_msgs/Image (如 /kinect2/sd/image_color_rect)
○深度图像：sensor_msgs/Image (如 /kinect2/sd/image_depth_rect)
○三维点云：sensor_msgs/PointCloud2 (如 /kinect2/sd/points)
●优点：同时提供颜色和深度信息，赋予机器人三维空间感知能力。
●缺点：结构相对复杂、成本较高、深度测量易受环境光干扰（尤其结构光和 ToF 方案）。
●用途：
○自主避障与导航：通过深度图或点云识别障碍物并规划路径；
○三维重建与环境建模：构建地图用于定位和导航；
○目标检测与追踪：结合颜色和深度信息，更鲁棒地识别和跟踪目标；
○物体抓取与交互：精确感知物体的三维位置和姿态，实现精准抓取。

2总结：

| 特性``     | 普通 RGB 相机 (Camera)``   | RGB-D 深度相机 (Kinect)``                            |
| ----------------- | --------------------------------- | ----------------------------------------------------------- |
| 核心能力`` | 二维彩色成像``             | 二维彩色成像 + 三维深度测量``                        |
| 深度信息`` | ❌无``                     | ✅有``                                               |
| 数据输出`` | sensor_msgs/Image``        | sensor_msgs/Image``+sensor_msgs/PointCloud2`` |
| 典型原理`` | 针孔成像``                 | 结构光 / ToF / 双目视觉``                            |
| 核心优势`` | 成本低、色彩好、分辨率高`` | 三维空间感知能力``                                   |
| 主要局限`` | 无法测距``                 | 易受环境光干扰、成本较高``                           |
| 典型应用`` | 人脸识别、颜色追踪``       | 避障、导航、三维建模、物体抓取``                     |
