# L3 机器人足球场仿真环境配置

## 1. 实验目标

围绕 `image_pkg` 完成以下实验内容：

1. 识别 Kinect2 彩色图像话题。
2. 控制四个彩色小球随机运动。
3. 使用 HSV 方法识别目标小球。
4. 控制机器人完成跟球。
5. 完成人脸识别、对准和自动拍照加分项。

## 2. 实验环境

- Ubuntu 20.04
- ROS Noetic
- 工作空间：`/home/zyz/catkin_ws`
- 主要包：`wpr_simulation`、`image_pkg`

## 3. 关键话题

| 类别 | 默认话题 | 消息类型 | 用途 |
|---|---|---|---|
| 彩色图像 | `/kinect2/sd/image_color_rect` | `sensor_msgs/Image` | HSV 取球、跟球、人脸检测 |
| 速度控制 | `/cmd_vel` | `geometry_msgs/Twist` | 控制机器人底盘运动 |
| 红球速度 | `/red_ball_vel` | `geometry_msgs/Twist` | 控制红球 |
| 绿球速度 | `/green_ball_vel` | `geometry_msgs/Twist` | 控制绿球 |
| 蓝球速度 | `/blue_ball_vel` | `geometry_msgs/Twist` | 控制蓝球 |
| 橙球速度 | `/orange_ball_vel` | `geometry_msgs/Twist` | 控制橙球 |

## 4. 推荐流程

### 4.1 构建

```bash
source /opt/ros/noetic/setup.bash
cd /home/zyz/catkin_ws
catkin_make
source /home/zyz/catkin_ws/devel/setup.bash
rospack find image_pkg
```

### 4.2 球场场景

```bash
roslaunch wpr_simulation wpb_balls.launch
```

### 4.3 HSV 取球

```bash
roslaunch image_pkg hsv_demo.launch
```

### 4.4 跟球

```bash
roslaunch image_pkg follow_ball.launch run_ball_motion:=true
```

### 4.5 单人脸场景

```bash
roslaunch wpr_simulation wpb_single_face.launch
roslaunch image_pkg face_capture.launch
```

## 5. 快速检查命令

```bash
rostopic list | grep -E "kinect2|ball_vel|cmd_vel"
rostopic hz /kinect2/qhd/image_color_rect
rostopic echo -n 5 /cmd_vel
```
